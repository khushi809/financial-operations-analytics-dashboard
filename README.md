# Financial Operations Analytics Dashboard

## Overview

This project provides an end-to-end Financial Operations Analytics solution using Python, Machine Learning, and Power BI.
The objective is to transform raw business data into actionable insights through forecasting, customer analytics, segmentation, and profitability analysis.

## Project Components

### 1. Data Cleaning & Exploratory Data Analysis (EDA)
* Cleaned and preprocessed retail and telecom datasets
* Handled missing values and duplicates
* Performed exploratory analysis using Pandas, Matplotlib, and Seaborn

### 2. Revenue Forecasting
* Built a time-series forecasting model using Prophet
* Predicted future revenue trends
* Generated revenue forecast reports

### 3. Customer Churn Prediction
* Built a Random Forest machine learning model
* Identified customers likely to churn
* Generated churn risk scores

### 4. RFM Customer Segmentation
* Calculated Recency, Frequency, and Monetary scores
* Classified customers into:

  * VIP
  * Loyal
  * At Risk
  * Regular

### 5. Profitability Analysis
* Identified top-performing customers
* Analyzed product profitability
* Evaluated country-level revenue performance


## Technologies Used
